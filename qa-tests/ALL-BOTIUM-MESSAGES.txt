  ┌─────────────┐
  │   Botium    │
  │   Runner    │

         │

         ↓ Goes to WhatsApp Web

  │ web.whatsapp.com │
  └──────┬───────────┘
         │
         ↓ Finds chat input box
  ┌──────────────────────────────┐

         ↓ Types message
  ┌──────────────────────────────┐

         │
         ↓ Message sent to bot
  ┌─────────────────────┐

💡 IMPORTANT:

Botium doesn't send HTTP requests to your API.
It literally types in WhatsApp Web like a human user!

This means:
  ✅ Tests the EXACT user experience
  ✅ Tests WhatsApp integration
  ✅ Tests full message flow
  ❌ Requires WhatsApp Web session
  ❌ Slower than API tests

Alternative for faster testing:
  - Use Jest unit tests (instant, no WhatsApp needed)
  - Use Message Simulator (direct API calls)
  - Use Playwright for dashboard (UI only)
